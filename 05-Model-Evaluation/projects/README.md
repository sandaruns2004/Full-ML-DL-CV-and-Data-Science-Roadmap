# 💻 Model Evaluation Projects

This directory contains standalone, end-to-end Jupyter Notebooks that test your ability to apply Model Evaluation techniques to realistic, noisy datasets.

## 🚀 Available Projects

1. `01-Model-Evaluation-Dashboard.ipynb`: Interactive UI to test thresholding.
2. `02-Classification-Metrics-Explorer.ipynb`: Investigating the "ROC Illusion".
3. `03-Cross-Validation-Laboratory.ipynb`: Measuring the variance of different CV splitters.
4. `04-Hyperparameter-Tuning-Engine.ipynb`: Speed and score comparison of exhaustive vs random search.
5. `05-Imbalanced-Data-Challenge.ipynb`: Fraud detection using class weights and SMOTE pipelines.
6. `06-End-To-End-Model-Deployment.ipynb`: Simulating production decay and generating Evidently AI drift reports.

## 🛠️ Instructions
Each project is contained entirely within its `.ipynb` file. Open the notebook, read the objectives, and execute the cells.
