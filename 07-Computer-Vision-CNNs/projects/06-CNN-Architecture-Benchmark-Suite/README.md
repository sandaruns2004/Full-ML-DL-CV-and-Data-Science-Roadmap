# 06 - CNN Architecture Benchmark Suite

## 🎯 Objective
Understand the progression of Convolutional Neural Networks by implementing and comparing historical architectures like LeNet against modern baseline custom CNNs.

## 🧠 Concepts Covered
- **LeNet Architecture**: Replicating Yann LeCun's pioneering 1998 CNN architecture.
- **CIFAR-10 Dataset**: Moving beyond MNIST to a dataset with color images (3 channels) and 10 complex classes (cars, birds, cats, etc.).
- **Benchmarking**: Evaluating parameter counts and comparing training times vs. accuracy.

## 🚀 Getting Started
This is a purely code-driven analytical project.

To run the benchmark suite:
1. Ensure you have Jupyter installed (`pip install jupyter`).
2. Navigate to the `notebooks/` directory.
3. Open `Benchmark_Suite.ipynb` and execute the cells to download the CIFAR-10 dataset and train the different models.
