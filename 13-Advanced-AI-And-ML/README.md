# 13 - Advanced AI and Machine Learning

Welcome to the **Advanced AI and ML** module! This is the capstone repository of the entire Machine Learning Roadmap. It bridges the gap between traditional AI education and the cutting-edge of modern AI research and production. 

Having mastered Machine Learning, Deep Learning, Computer Vision, and Natural Language Processing, this module takes you into the systems that power modern Artificial General Intelligence (AGI) efforts: Reinforcement Learning, Multi-Agent Systems, RAG, and Efficient Model Optimization.

## 💻 Educational Notebooks

In the `notebooks/` directory, you will find interactive Jupyter Labs designed to visually demonstrate complex concepts:

1.  **Representation_Learning_Lab.ipynb**: Visualize High-Dimensional Embeddings using PCA and t-SNE.
2.  **Contrastive_Learning_Lab.ipynb**: A miniature implementation of SimCLR to understand positive and negative pairing.
3.  **Reinforcement_Learning_Playground.ipynb**: Train a Q-Learning agent to beat a simple Gymnasium environment.
4.  **RAG_System_Lab.ipynb**: Build a localized Vector Database from scratch using FAISS and SentenceTransformers.
5.  **Agent_Workflow_Lab.ipynb**: Write custom Python tools and trigger them using LLM Function Calling.
6.  **Model_Optimization_Lab.ipynb**: Quantize an FP32 model down to INT8 and measure the inference speedup.
7.  **AI_Evaluation_Lab.ipynb**: Implement the LLM-as-a-Judge pattern to score generated text.

Because this module is the culmination of the entire roadmap, it is highly theoretical and abstract. Do not rush through the theory. Spend time reading the provided external links in the `.md` files, review the official PyTorch documentation for complex architectures, and most importantly, practice reading the primary research papers discussed in Lesson 14.
