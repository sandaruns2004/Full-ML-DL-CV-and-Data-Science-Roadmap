# 📚 00 - Prerequisites

Welcome to the **Prerequisites Module**! Before diving into Deep Learning architectures, building complex Machine Learning pipelines, or dealing with MLOps deployments, you must build an indestructible foundation. 

This module serves as a comprehensive "bridge course" to get you up to speed with the AI ecosystem, computer fundamentals, programming, tooling, and mathematics required for the rest of this roadmap.

---

## 🎯 Learning Outcomes

By the end of this module, you will be able to:
- Clearly distinguish between AI, ML, DL, DS, NLP, CV, and MLOps.
- Understand how your computer's hardware (CPU, GPU, RAM, VRAM, Storage) interacts during data processing and model training.
- Write robust, PEP-8 compliant Python code utilizing OOP principles and type hinting.
- Set up a professional ML development environment using VS Code, Conda, and Jupyter.
- Confidently use Pandas, NumPy, Matplotlib, Seaborn, and SciPy for data manipulation and exploratory data analysis.
- Collaborate on code and resolve merge conflicts using Git and GitHub.
- Navigate cloud instances, manage long-running background processes, and perform file operations using Linux terminal commands.
- Understand the mathematical intuition behind Linear Algebra, Calculus, Probability, and Statistics driving modern algorithms.
- Extract, join, and aggregate large datasets using SQL window functions and CTEs.

Please go through these files in the following order:

    *   Understand the difference between AI, ML, DL, Data Science, NLP, and CV.

    *   Learn about CPUs, GPUs, RAM, VRAM, Storage, and Data Pipelines.
3.  **[03-Programming-Fundamentals.md](./03-Programming-Fundamentals.md)**
    *   Review core Python concepts like control flow, functions, and OOP.
4.  **[04-Development-Environment-Setup.md](./04-Development-Environment-Setup.md)**
    *   Set up Miniconda, VS Code, and Jupyter Notebooks like a professional.
5.  **[05-Python-Essentials.md](./05-Python-Essentials.md)**
    *   Master NumPy, Pandas, Matplotlib, Seaborn, and SciPy. Includes extensive, runnable code snippets.
6.  **[06-Git-And-GitHub.md](./06-Git-And-GitHub.md)**
    *   Learn how to version control your code, manage branches, and collaborate using Pull Requests.
7.  **[07-Linux-Fundamentals.md](./07-Linux-Fundamentals.md)**
    *   Navigate headless cloud servers, manage background training processes (`nohup`, `tmux`), and check GPU usage.
8.  **[08-Mathematical-Foundations.md](./08-Mathematical-Foundations.md)**
    *   Review Linear Algebra, Calculus, and Optimization Theory. Visualized with Matplotlib.
9.  **[09-Probability-And-Statistics.md](./09-Probability-And-Statistics.md)**
    *   Review Bayes' Theorem, Probability Distributions, Law of Large Numbers, CLT, and MLE.

## 🛠️ Mini-Projects

Once you have reviewed the content, test your skills with these practical mini-projects located in the `projects/` directory:

*   **[03-Environment-Setup-Validation.py](./projects/03-Environment-Setup-Validation.py)**: Run this script first to ensure your Python (3.10+), Git, Jupyter, and ML library installations are correct.
*   **[01-Python-Data-Analysis-Starter.ipynb](./projects/01-Python-Data-Analysis-Starter.ipynb)**: Load a synthetic e-commerce dataset, handle missing values, perform EDA, and generate rich Seaborn plots.
*   **[02-Git-Workflow-Simulation.md](./projects/02-Git-Workflow-Simulation.md)**: Practice creating feature branches, simulating a realistic merge conflict between a Random Forest and XGBoost model, and resolving it.
*   **[04-Linux-Command-Toolkit.sh](./projects/04-Linux-Command-Toolkit.sh)**: Use this handy bash script to scaffold a standard ML project directory, check system resources, and generate mock CSV data on any Linux machine.

## ⚡ Quick Start

1. Open your terminal.
2. Navigate to this directory.
3. Activate your environment (`conda activate ml-roadmap`).
4. Run the validation script:
   ```bash
   python projects/03-Environment-Setup-Validation.py
   ```
5. If everything passes, you are ready to begin the **Prerequisites Module**!
