# 04 - Employee Attrition Forecast

## 🎯 Objective
Model tabular HR data to predict employee turnover, demonstrating how deep neural networks can handle categorical embeddings.

## 🧠 Concepts Covered
- **Categorical Variables**: Handling non-numerical data in Neural Networks (One-Hot Encoding vs. Entity Embeddings).
- **Interpretability**: Analyzing the weights of a neural network to understand what features (e.g., salary, department) drive attrition.

## 🚀 Getting Started

This project is structured as a purely analytical Jupyter Notebook.
1. Ensure you have Jupyter installed (`pip install jupyter`).
2. Navigate to the `notebooks/` directory.
3. Open `Attrition_Modeling.ipynb` to step through the data preparation, neural network training, and weight interpretation process.
