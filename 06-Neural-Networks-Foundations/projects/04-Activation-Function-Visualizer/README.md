# 🌊 Project 04: Activation Function Explorer

## 🎯 Goal
Build an interactive educational tool to visualize mathematical non-linearities and their derivatives.

## 📝 Description
Understanding the geometry and calculus of activation functions is crucial for diagnosing issues like Vanishing Gradients. This project tasks you with creating a cleanly formatted Jupyter Notebook or Streamlit application that charts these functions.

## ✅ Requirements
- Implement the mathematical formulas (and their derivatives) for: Sigmoid, Tanh, ReLU, Leaky ReLU, and GELU.
- Create a Python script that plots these functions side-by-side.
- The charts must include clear axes, legends, gridlines, and clearly demonstrate where the derivatives flatline (approach zero).

## 📁 Files
- `activation_explorer.ipynb`
- `utils/math_functions.py`
