# 02 - Telecom Churn Predictor

## 🎯 Objective
Use a Feed-Forward Neural Network to predict whether a telecom customer will cancel their service (churn) based on their account data.

## 🧠 Concepts Covered
- **Tabular Data Processing**: Normalizing numerical features with `StandardScaler`.
- **Exploratory Data Analysis (EDA)**: Visualizing class imbalance before modeling.
- **Binary Classification**: Using `BCELoss` and a final Sigmoid activation.
- **Evaluation**: Interpreting final loss metrics in a pure data science workflow.

## 🚀 Getting Started

This project is structured purely as a Data Science analytical notebook.
1. Ensure you have Jupyter installed (`pip install jupyter`).
2. Navigate to the `notebooks/` directory.
3. Open `Churn_Analysis_and_Modeling.ipynb` to step through the data generation, EDA, and neural network training process.
