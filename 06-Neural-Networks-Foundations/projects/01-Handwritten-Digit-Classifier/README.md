# 🖊️ Project 01: Handwritten Digit Classifier

## 🎯 Goal
Implement a Neural Network capable of recognizing handwritten digits from the famous **MNIST dataset**. 

## 📝 Description
The MNIST dataset is the "Hello World" of Deep Learning. It consists of 70,000 grayscale images of handwritten digits (0-9).

In this project, you will build the exact same model using three different approaches:
1. **The Hard Way (NumPy):** To prove you understand the math.
2. **The Industry Way (PyTorch):** To learn the standard research framework.
3. **The Production Way (TensorFlow/Keras):** To learn the standard deployment framework.

## ✅ Requirements
- Download the MNIST dataset.
- Implement a Neural Network with at least 1 hidden layer.
- Train the model to achieve > 95% accuracy on the test set.
- Compare the training speed, ease of implementation, and final accuracy across NumPy, PyTorch, and TensorFlow.

- `tensorflow_classifier.py`
